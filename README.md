# Norte Estofados

Landing page da Norte Estofados — tapeçaria automotiva e náutica. Construído com Next.js e Tailwind CSS.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Configuração

- `lib/site-config.ts` — telefone, WhatsApp, endereço, e-mail, CNPJ e redes sociais.
- `lib/gallery-data.ts` — fotos e categorias do catálogo/galeria (atualmente placeholders do Unsplash).
